
# Changelog

## v3.5.0 - Production Candidate
*   **Feature**: Added NSIS Installer build configuration.
*   **Feature**: Added Portable ZIP build configuration.
*   **Feature**: Implemented File-Based Logging System (in `AppData/Roaming/1Man1Machine/logs`).
*   **Feature**: Added **Emergency Reset Combo** (L1 + R1 + SHARE) to kill all inputs instantly.
*   **Stability**: Added Single Instance Lock to prevent multiple app copies.
*   **Stability**: Added Global Exception Handler to prevent crashes on unhandled errors.
*   **UX**: Wired up "View System Logs" button in Analytics view.
*   **Fix**: Hardened Gamepad Polling loop against disconnects.
